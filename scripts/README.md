# Scripts

One-off and maintenance scripts (CI, release, security, Hugging Face, GIF/demo, etc.). **Not part of the installed `bleu-js` package.** Run only when needed; many require specific env (e.g. AWS, HF token). For release and publish, see [docs/RELEASE_CHECKLIST.md](../docs/RELEASE_CHECKLIST.md) and [docs/PYPI_PUBLISH.md](../docs/PYPI_PUBLISH.md).
