# Internal docs (maintainer-only)

This folder holds **maintainer-only** documentation that is **not** included in the public repo. The folder is in `.gitignore` except for this README.

**Contents (local only):**

- `GITHUB_RELEASE_INSTRUCTIONS.md` — How to create GitHub releases
- `BLEUJS_ORG_CHECKLIST.md` — bleujs.org content checklist
- `WHATS_NEXT.md` — Internal priorities and checklist
- `SECURITY_FIXES_2025.md` — Security fix log
- `COMPARISON_LAST_MONTH.md` — Internal comparison notes

If you are a maintainer, copy these from another maintainer or from your local clone. Do not add them to the public repository.

Public docs (install, API, contributing, security policy, etc.) remain in `docs/` and the repo root.
