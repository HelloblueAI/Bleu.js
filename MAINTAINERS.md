## GitHub contributor graph

GitHub’s automatic **Contributors** avatars on the repo homepage may include **bots and automation** (e.g. Dependabot, GitHub Actions, historical Cursor agent commits). Those accounts are **not** maintainers.

For ownership and review, use this file and [.github/CODEOWNERS](.github/CODEOWNERS).

## Becoming a maintainer

Maintainer status is by invitation after sustained, high-quality contributions. Start with [CONTRIBUTING.md](CONTRIBUTING.md) and [good first issues](https://github.com/HelloblueAI/Bleu.js/issues?q=is%3Aopen+label%3A%22good+first+issue%22).
